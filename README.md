# Data Structures Projects

##  Overview
This repository contains multiple Java projects developed as part of coursework and practice in **Data Structures and Algorithms**.  
Each project demonstrates a different concept, ranging from custom list implementations to games and system simulations.

---

##  Projects

### 1. CallCenterSystem
A simulation of a call center system, demonstrating queue management and scheduling logic.

### 2. CustomArrayList_DataStructures
A custom implementation of an `ArrayList` in Java, including sorting and recursion to find maximum elements.

### 3. Linked_List
Implementation of a singly linked list with core operations such as insertion, deletion, and traversal.

### 4. Word_Game
A word-based game project showcasing string manipulation, arrays, and algorithmic logic.

---

##  Technologies Used
- Java (JDK 8+)
- Object-Oriented Programming concepts
- Data Structures (Arrays, Linked Lists, Queues, Recursion)
